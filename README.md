# Складмяк

## Отбор товара
Отбор товара происходит по принципу FIFO.

## Система статусов ячеек
⦁	Свободно для брони на размещение\
⦁	Забронировано для размещения\
⦁	Свободно для брони на отбор\
⦁	Забронировано для отбора\
⦁	Забронировано для инвентаризации\
